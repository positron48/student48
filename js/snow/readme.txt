Как одно из украшательств вашего сайта в зимний период, может послужить красивый скрипт, отображающий летящие снежинки, такие как на этом сайте.

Скрипт создан с помощью javascript, поддерживает формат png и gif для отображения снежинки, по умолчанию содержит только png изображения снежинок.

Есть возможность заставить снежинки реагировать на движения "мышкой". Снег собирается внизу экрана и лежит там какое то время :) Так же есть возможность отключить эту функцию. Содержит ещё множество параметров

Этот скрипт не является ни модулем, ни плагином, ни каким либо ещё расширением для каких либо CMS, он независимый. Хоть это и может заставить немного повозиться с кодом пользователей CMS, установить код себе на сайт думаю сможет каждый, даже начинающий web-разработчик
Установка скрипта падающего снега на сайт
Качаем архив (содержит сам скрипт, изображения снежинок в формате png и небольшой readme файл с этой инструкцией
Из архива достаём папку snow и загружаем на сервер в корневую директорию сайта
В сайт, после тега <body> вставляем код: 

<script type="text/javascript">
sitePath = "/";
sflakesMax = 64;
sflakesMaxActive = 64;
svMaxX = 3;
svMaxY = 3;
ssnowStick = 1;
sfollowMouse = 1;
</script>
<script type="text/javascript" src="/snow/snow.js"></script>
 
проверяем

Если скрипт снега устанавливается на сайт с CMS Joomla! 1.5, то можно применить такую схему:

<?php
if(JHTML::Date( 'now', '%m' )=='12' OR JHTML::Date( 'now', '%m' )=='1' OR JHTML::Date( 'now', '%m' )=='2') : ?>
<!-- Снег -->
<script type="text/javascript">
sitePath = "/";
sflakesMax = 64;
sflakesMaxActive = 64;
svMaxX = 3;
svMaxY = 3;
ssnowStick = 1;
sfollowMouse = 1;
</script>
<script type="text/javascript" src="/snow/snow.js"></script>
<!-- Снег end -->
<?php endif; ?>
 

Этот код узнаёт текущий месяц средствами Joomla, делает проверку и включает или отключает скрипт.
В данном случае снег будет падать 3 зимних месяца.
